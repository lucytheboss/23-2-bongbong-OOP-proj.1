//READ ME

The program is composed of 4 files. 
1. main.cpp
2. header.h
3. Student.h
4. insertion.cpp


code runs by playing in main.cpp file, using ctrl+alt+N at vsc for windows laptop and control + option + N for MacBook

When the program runs, you can visit the main menu and insert, search, sort or list all.
Insertion menu is where you insert new information of student
With searching option, you can search for students based on their information (name, department, admission year etc)
With sorting option, you can get list of student ordered in name, admission year, and etc.

As to conclude, the program is a program that helps you get, sort, and view student information you've inserted.
